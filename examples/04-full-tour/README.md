# 04 - full tour

Every command type, every say mode, every say align, every timer style, a
choice, `goto`, `gotoScript` (multi-line comment handled by the flatten
pass), `eval`, `enable`/`disable`, notifications with and without ids, and
audio playback with both `file()` and `file.pattern()`. Mirrors the shape
of a full platform export.

## Run

```bash
bun run examples/04-full-tour/build.ts
```

Produces `out.json` next to the story.
